# 2.切题四件套 #
	1.与面试官沟通题目详细信息，确保对题目理解没有问题。
	2.尝试列举出所有的问题解决方案。
	3.比较时间和空间复杂度，选择最优解。
	4.测试用例。



##3.五毒神掌  ##
	1.阅读题目，如果超过10分钟仍没有思路，直接去看答案，并且尝试记住。
	2.尝试自己去写答案（过程中不能看答案），并进行单点调试。
	3.第二天再次尝试去写此题。
	4.这一周再次尝试解答此题。
